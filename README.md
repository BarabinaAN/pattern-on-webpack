# Шаблон для многостраничного сайта

### Конфигурация включает поддержку
* CSS
* SCSS
* расстановку autoprefixer
* Jquery
* ES6+
* slick slider
* lodash
* уменьшение размера css, js

### Версия NPM
14.17.3

### Работа с проектом
Установка:
```
npm i
```

Запуск:
```
npm start
```

Сборка версиии проекта production:
```
npm run build
```

Сборка версиии проекта developer:
```
npm run dev
```
